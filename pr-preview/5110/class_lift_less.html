Detailed Description

template<class T>
class LiftLess< T >

Lifts < on objects to functions and functors for ==, !=, <, >, <=, and >= on pointers, and also derives ==, !=, >, <=, and >= on objects. The lifted virtual operator< is assumed to implement a strict weak ordering relation.
