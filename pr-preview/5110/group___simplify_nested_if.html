class P4::DoSimplifyComplexCondition

Pass that tries to simplify the conditions to a simple comparison of constants.

Tofino does not support complex condition on if statements in deparser. This pass tries to simplify the conditions to a simple comparison to constant, e.g.: 

if (intrinsic_md.mirror_type == 1) {}
