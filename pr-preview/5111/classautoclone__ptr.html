SPDX-License-Identifier: Apache-2.0 Adaptor/wrapper class for std::unique_ptr that allows copying by cloning the pointed at object. An autoclone_ptr can be used when you want to make a pointer field act like a value field – any copy of the containing object will deep-copy all the autoclone_ptr-referenced objects avoid any accidental shallow copy aliasing or slicing (particularly useful for dynamic classes).
