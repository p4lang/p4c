BFN::RemoveSelectBooleans
Midend

Set of passes that convert boolean values in select statements to bit<1> values. The node is also transformed to IR:BFN:ReinterpretCasts.

These passes take care of select statements with boolean values such as this one:

(...)

header example_bridge_h { bit<32> dst_mac_addr_low; bit<32> src_mac_addr_low; bool ipv4_valid_in_ingress; }

struct metadata_t { example_bridge_h example_bridge_hdr; }

(...)

transition select(eg_md.example_bridge_hdr.ipv4_valid_in_ingress) { true: accept; default: reject; }

Expression "eg_md.example_bridge_hdr.ipv4_valid_in_ingress" is translated to "(bit<1>)eg_md.example_bridge_hdr.ipv4_valid_in_ingress", select value "true" is translated to (bit<1>)1 while the associated nodes are transformed to IR::BFN::ReinterpretCast.
