◆ ixbar_imbalanced_alignment()

int PHV::v2::GreedyTxScoreMaker::ixbar_imbalanced_alignment ( const ordered_set< ContainerByte > & cont_bytes)
static
Returns
an score of how imbalanced container bytes are. Imbalance is measured under a mod-4 system. For example, ixbar bytes layout 0 1 2 3 H2@0 MH5@1 W6@2 W6@3 H9@0 MH5@1 The score is 2 because there are 2 ixbar bytes that has more than minimal bytes (1 in this case). See gtests.
