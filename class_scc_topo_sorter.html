◆ scc_topo_sort()

std::unordered_map< int, int > SccTopoSorter::scc_topo_sort ( ) const

run SCC and topological sort on the internal graph, returns a mapping from node_id to toposort index. For example, Assume a <- b represents a depends on b, and we have a <- b <- c <-> e ^ | d will be converted to, a <- b <- {c, e} ^ | d and the output will be, d, c, e: 1 b: 2 a: 3
